Mobile App Workforce Management

Hero Clock-in/out App

Hero Clock-in/out App is an Android workforce app for pest control teams. Technicians clock in and out with GPS-backed verification, see daily schedules with restaurant assignments, and file service tickets with photo documentation. The system is built offline-first, so work continues when connectivity is spotty and syncs when the network returns. A companion admin view supports attendance monitoring and operational oversight. Built with Java in Android Studio, using Firebase, MySQL, and device GPS.

Clock in / clock out — Time tracking with real-time status and GPS verification.
Schedule — Daily schedules with restaurant (site) assignments.
Tickets — Service tickets with photo uploads for documentation.
Location — Location-aware workflows tied to assignments and verification.
Offline-first — Core actions work without a constant connection; data syncs when online.
Admin / reporting — Dashboard-style monitoring of attendance and workforce activity.

Mobile App Loyalty System

Engr's Brew

Engr’s Brew is a native Android loyalty application built for a neighborhood café. It connects the counter experience to a digital wallet: customers don’t need to order online to earn rewards—they show a unique QR code in the app when they pay in person. Store staff use a dedicated admin scanner to read that code, enter the purchase amount, and credit points immediately. Behind the scenes, Firebase Realtime Database keeps balances and activity in sync so customers see updates without refreshing the app. The app is organized around everyday loyalty tasks: viewing your QR, checking points and history, browsing rewards and tiers, and managing profile settings.

QR-based loyalty — Earn points from physical, in-store purchases; no online order required to participate.
Admin scanner panel — Staff-facing flow to scan customer QR codes and enter purchase amounts to award points
Real-time updates — Points and related data sync with Firebase Realtime Database for instant balance and activity updates.
Points & history — Track earnings and see how points change over time.
Rewards — Browse and redeem rewards tied to the loyalty program.
